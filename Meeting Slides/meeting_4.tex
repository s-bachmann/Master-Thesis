\begin{frame}{Overview: ToDos of last Meeting}
    \begin{itemize}

        \item Plot x and y components of space-time plots
        \item Source-time function in x and y 
        \item Wiggle plots of traces
        \item Run supershear simulation to check if it's different from sub-Rayleigh
        \item Make domain bigger but keep simulation time to get rid of ABC effects
        \item Summary of crack types
    \end{itemize}

\begin{frame}{Summary of Model}
    \begin{itemize}
        \item Implemented cohesive zone model described by \cite{mahajanModelingInterfacialCrack2008}
        \item Implemented functions described by \cite{mahajanModelingInterfacialCrack2008}
        \item Accounts for cohesive zone (weak layer)
        \item Focuses primarily on mode II cracks 
        \item Two-layered model: snow and air
        \item Fault mechanism: 10Hz moment tensor sources
        \item Forces of sources are computed by using a fraction of crack modes: eg mxy is calculated by total force times fraction of mode II
        \item Sources every 2m 
    \end{itemize}
\end{frame}
